# Frontend - Fitness Expert System

## Instalación

```bash
cd frontend
npm install
```

## Ejecutar

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

## Build

```bash
npm run build
```

## Tecnologías

- React 18
- TypeScript
- Vite
- NextUI
- TailwindCSS
- Axios
